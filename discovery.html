Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:
innerHTML: Set the innerHTML property for add or change the HTML content inside an element.
innerText: InnerText is that the text exactly as we see it on the screen. It does not include HTML tags and skips any text that is hidden with CSS styles.
textContent: The textContent property ignores all HTML tags and gives only the text inside an element. The innerText property shows text as it appears on the screen, while textContent shows text as it is written in the HTML code.

Question-3: What is event delegation in the DOM?

Answer:
Event delegation means adding one event listener to a parent element instead of many on each child. When something inside the parent is clicked, the parent handles it.

Question-4: What is event bubbling in the DOM?

Answer:
Event bubbling is when an event starts on a child element and then moves up to its parent elements. This lets parent elements handle events for their children, making it easier to manage events.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:
Creating, adding, and deleting new elements with JavaScript is very easy. First, a new element is created using document.createElement() . Then, text is added to that element using textContent or innerHTML . To add the new element, appendChild() is used. This allows you to add a new element to the parent element, for example, parentElement.appendChild(newElement) can be used to add a new element to the parent element.
